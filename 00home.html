The vehicle is an autonomous agent defined and used accordingly to the class Vehicle. The class consists of several methods:
-Update that modifies parameteres of the vehicle,
-ApplyForce, which applies new vectors(forces) to the vehicle,
-steering behaviours (listed in the left column).
In the box on the right you can see basic methods that are used throughout simulations thus, they won't appear on other pages. 
class Vehicle {
  constructor(x, y) {
    this.acceleration = new Vector(0, 0),
    this.velocity = new Vector(0, 0),
    this.position = new Vector(x, y),
    this.radius = 30,
    this.maxspeed = 4,
    this.maxforce = 0.1;
    this.desiredVelocity = new Vector(0, 0);
  }
  update() {
    this.velocity = this.velocity.add(this.acceleration);
    this.velocity.limit(this.maxspeed);
    this.position = this.position.add(this.velocity);
    this.acceleration = this.acceleration.multiply(0);
  }
  applyForce(force) {
    this.acceleration = this.acceleration.add(force);
  }
}
