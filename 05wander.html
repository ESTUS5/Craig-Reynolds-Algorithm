  Wander algorithm simulates random movement of the agent. In order to achive desired movement the random number theta is created outside of method. Theta is a random degree (from 0 to 360). Inside the Wander method theta is modified by random number of certain length (in this example it's range is from -15 to 15). Then new theta is used to calculate points on circle with certain radius. Calculated position is returned by the Wander and passed to the Seek. Thanks to small steps vehicle will be able to move ahead quite smoothly and randomly.
  The orange vector pointing out of the center of vehicle is a visualization of desired velocity and the violet one shows current velocity of the agent. The radius of circle and vector pointing to that circle can be modified.

wander(length = 30, radius = 25) {
  var lengthVector = this.velocity.normalize();
  lengthVector = lengthVector.multiply(length);
  lengthVector = this.position.add(lengthVector);
  this.theta = this.theta + ((Math.random() * 15) - (15 * 0.5));
  var radiusVector = new Vector(Math.cos(this.theta) * radius,
  Math.sin(this.theta) * radius);
  radiusVector = lengthVector.add(radiusVector);
  return radiusVector;
}
