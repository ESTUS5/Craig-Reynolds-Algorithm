  The Unaligned Collision Avoidance is one of the approaches to avoid moving objects. In this example other vehicles are cause of collisions. To prevent this thier predicted positions are compared with the vehicle's. Result tells if avoidance has to be performed. In order to avoid collision the vehicle will accelerate or decelerate and seek towards the opposite direction.
  The orange vector pointing out of the center of Vehicle is visualization of the desiredVelocity and the violet one shows current velocity vector. White vector conecting two vehicles occurs when they are on collision path. Red vector is predicted velocity and green represents avoiding force.

unalignedCollisionAvoidance(Vehicles,futurePos=0){
    var predictedPosition = this.velocity.normalize().multiply(futurePos);
    predictedPosition = predictedPosition.add(this.position);
    for (let i = 0; i < Vehicles.length; i++) {
      if(Vehicles[i].position.x != this.position.x &&
         Vehicles[i].position.y != this.position.y
        ){
        var avoidPosition = Vehicles[i].velocity.normalize().multiply(futurePos);
        avoidPosition = avoidPosition.add(Vehicles[i].position);
        if(predictedPosition.distanceBetweenPoints(avoidPosition) < this.radius*2){
          var avoid = avoidPosition.mirror(this.position);
          if(this.velocity.length() > (this.maxspeed/2)){
            avoid = avoid.add(this.velocity.multiply(-0.5));
          }
          else if(this.velocity.length() <= (this.maxspeed/2)){
            avoid = avoid.add(this.velocity.multiply(0.5));
          }
          return avoid;
        }
      }
    }
    return new Vector(0,0);
  }
